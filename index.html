Отчёт по сайту М.Видео

Структура сайта

    Основной контент сайта (если не учитывать модальные окна, скрипты, таблицы стилей и т.п.) содержится в теге wrapper.

  1. header
    • header-fixed
      • header-top-line
      • header-main
    • header-top-categories (множество ссылок в контейнере flex)
    • header-catalog
  2. main
    Однотипные блоки со слайдерами

    Предложение подписаться
  3. footer (деление на столбцы - flex, внутри - список ul>li)

Реализация блока "brands-section"

  1. Создаём блок 
    <div class="brands-section">...</div>
  2. В этот блок вставляем заголовок 
    <div class="section brands-section__header">Популярные бренды</div>
  3. Добавляем контейнер для содержимого
    <div class="fl-popular-brands"> ... </div>
  4. Внутри - тег маркированного списка
    <ul class="fl-popular-brands__content" style="position: relative; left: 0px;"> ... </ul>
  5. Добавляем элемент маркированного списка 
    <li class="brand-item"> ... </li>
  6. Вставляем ссылку, чтобы изображение внутри было кликабельно (активно для нажатия)
    <a href="/apple?from=homepage" class="brand-item__link"> ... </a>
  7. Добавляем контейнер для изображения
    <div class="lazy-load-image-holder lazy-loaded"> ... </div>
  8. Вставляем изображение с логотипом бренда
    <img alt="Apple" class="lazy" src="//static.mvideo.ru/media/Promotions/Promo_Page/2020/September/brands/1/Apple.png">
  9. Для правильного позиционирования картинок используем "flex" для "ul" (класс "fl-popular-brands__content")
    display: flex;
  10. Скрываем вертикальное вымещение 
    overflow-y: hidden;
  11. Добавляем перенос дочерних элементов
    flex-wrap: wrap;
  12. Добавляем максимальную высоту
    max-height: calc(8.4rem * 2 + 2rem);
  13. Убираем маркировку у списка
    list-style: none;
  14. Добавляем размеры и "margin" для "li"
    width: 10.7rem;
    height: 5.2rem;
    margin-right: 1.2rem;
    margin-bottom: 1.2rem;
    position: relative;
  15. Добавляем "padding" для контейнера изображения
    padding: .8rem;
  16. Тегу "li" присваиваем высоту
    line-height: 17px;
  17. Тегу "a" добавляем фоновый цвет (серый)
    background: #f5f5f6;

Список ресурсов для изучения верстки

  1. Bouharstev Studio

    Влог "Заметки программиста" пока не содержит видео на эту тему. Но планируется сезон про веб-разработку. Лично для меня это дополнительный стимул к изучению и практике.

  2. BrainsCloud

    YouTube-канал с обучающими видео о вебе. Интересно и практикоориентрованно.

  3. Фрилансер по жизни

    Отличный канал о веб-разработке и удалённой работе. Изучил создание многих элементов и приёмов.

  4. loftblog

    Подписался после рекомендации преподавателей. Пока не успел достаточно изучить.

  5. Web Dev Simplified

    Подписался по рекомендации однокурсников. Вроде, неплохой.

  6. MDN

    Содержит хороший и полный справочник по css и html

  7. htmlbook.ru

    Грамотно работает SEO, долгое время выпадал первым в поисковых запросах. Хорош в качестве вспомогательного инструмента, но дизайн не очень.

  8. html5book

    Достаточно неплохие обучающие статьи по веб-разработке. Полезен для пошагового обучения.

  9. WebReference

    Подробно расписаны стили и тэги, есть примеры и даже местами песочница.

  10. Центр Google Поиска

    Обучение SEO. О том, как индексирует Google.

  11. Вебмастер.Справка

    Аналогично - от Яндекса.

  12. CSS-Tricks

    Много статей, гайдов и туториалов по CSS. Очень хороший сайт для продвинутого изучения.

  13. CodePen

    Много отличных примеров разных работ. Можно не только искать решать проблемы, но и вдохновляться приёмами и идеями.
